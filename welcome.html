Getting Started

Simply type 'omni' in Chrome's search bar, then a space, and type in the URL you want to view. OmniProxy will then open the URL in the selected proxy in the current tab.

In some cases, OmniProxy modifies the functionality of the site you use as a proxy. If you have problems using those sites normally you should temporarily disable OmniProxy.

It's easy to change the proxy OmniProxy uses. Go to Options to switch between all available proxies.
