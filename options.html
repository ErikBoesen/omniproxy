Choosing a proxy

You can change the proxy that OmniProxy uses if one does not work or has unintentional side effects.

Google Translate: This is the best option for most people, it works on the most networks but can have errors on some web page layouts. This is the default selection. OmniProxy automatically hides its language bar.

Google Modules: This is part of the discontinued iGoogle service and works the best (and even allows you to download files), but may be blocked on some networks.

Google Web Cache: This is the cache used for Google Search. Not available for all sites, and content can be anywhere from a few hours to a few days old. OmniProxy hides Web Cache's info bar.

Anonymouse: This is a web-based proxy designed to anonymously browse the internet.

ProxyWeb: This is another web-based proxy. OmniProxy automatically hides its popup advertisements and options bar.

How to use

Simply type 'omni' in Chrome's search bar, then a space, and type in the URL you want to view. OmniProxy will then open the URL in the selected proxy in the current tab.

In some cases, OmniProxy modifies the functionality of the site you use as a proxy. If you have problems using those sites normally you should temporarily disable OmniProxy.

OmniProxy is open-source under the GPLv3 license on GitHub. You can browse the code, help fix bugs, or even make your own OmniProxy!
