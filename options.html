Choosing a proxy

You can change the proxy that OmniProxy uses if one does not work or has unintentional side effects.

Google Translate: This is the best option for most people, it works on the most networks but can have errors on some web page layouts. This is the default selection.

Google Modules: This is part of the discontinued iGoogle service and is primarily used for hosting gadgets. This works the best (and even allows you to download files), but may be blocked on some networks.

Google Mobilizer: This is a discontinued service by Google that renders web pages for simple web browsers. Pages are rendered with almost all CSS styling removed.

Google Web Cache: This is a cache used for Google Search.
